How to Use?


swagger UI is here

Search Songs

        
GET

        /search-songs?query[song name]&token=YOUR_TOKEN&limit=[number of songs you want to get] 

        example: /search-songs?query=daylight&token=TOKEN&limit=10
      
        
Output

        [{
          "song_id": "12345678901012131314151617181920",
          "images": ["image-url-1", "image-url-2"],
          "title": "song full title"
        }]

Download song by id

        
GET

        /download-song-by-id?id=[song id that you get from /search-songs page]

        example: /download-song-by-id?id=380750890760670080911221380760840
        
          
Output

          you will get listenable song. You will get the Content-Type from headers, most often it is webm

Get song's subtitles

        
GET

        /get-subtitles/[song id]

        example: /get-subtitles/380750890760670080911221380760840
        
          
Output

          [{'en': "song lyrics", "fr": "song lyrics in french"...}]

Get Song's clip

        
POST

        /get-clip/[song id]

        example: /get-clip/380750890760670080911221380760840 (POST method only)
        
          
Output

          {
            "mime_type": "video/mp4",
            "url": "https://songfinder.alwaysdata.net/download-clip/download/UFRCVVZYY3dSV0pPVG1wWlNFWlVZbFprUlZwWlJqRk5XakZYVmxVMWExWldjRWhhZFZaSFZHOW9i..."
          }

Recognize Song from audio

        
GET

        /recognize (send the audio as file)

        example code in python: requests.get("http://127.0.0.1:5000/recognize", files={"audio": open("skyfall.mp3", 'rb')}) (in {"audio": open("skyfall.mp3", 'rb')} you have to use 'audio' word)
        hint: ask chatgpt or any AI like LLM to convert this into the programming language you know if you don't know how to wirte this in the programming languge you want use
      
        
Output

        {
          "results": [
            "id": "523981e5-0...",
            "recordings": [
              "artists": [
                {
                  "id": "073dae8a-4...",
                  "joinphrase": "...",
                  "name": "Eminem"
                },
                ...
              ]
            ]
          ]
        }
